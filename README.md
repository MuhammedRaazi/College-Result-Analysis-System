# College Result Analysis System

An interactive academic result analysis system built using Excel, Pivot Tables, Pivot Charts, Dashboards, and VBA.

## Project Overview

This project was inspired by the raw result workbook I received after completing my engineering degree. The original workbook only contained register numbers, subject codes, grades, and separate worksheets for departments without any analytics or visualization.

To improve the usability and analysis of academic data, I transformed the raw dataset into a complete result analysis system with dashboards, department-wise portals, KPI tracking, and VBA-based navigation.

---

## Features

* Interactive Dashboard
* KPI Cards
* Department-wise Result Analysis
* Pass vs Fail Comparison
* Department-wise Arrear Analysis
* Most Failed Subject Insights
* Department-wise Student Portals
* VBA-based Navigation System
* Structured Result Database
